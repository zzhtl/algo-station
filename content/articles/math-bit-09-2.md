# 判断 2 的幂：数学与位运算训练题解

一个正整数如果是 `2` 的幂，它的二进制表示只有一个 `1`：

```text
1  -> 0001
2  -> 0010
4  -> 0100
8  -> 1000
```

而 `n-1` 会把这个 `1` 变成 `0`，并把右侧低位全部变成 `1`。所以：

```text
n & (n-1) == 0
```

## Go 参考实现

```go
package main

func isPowerOfTwo(n int) bool {
	return n > 0 && n&(n-1) == 0
}
```

## 为什么必须判断 n > 0

`0 & -1 == 0`，如果不加 `n > 0`，会把 `0` 误判为 `2` 的幂。负数在补码下也可能满足某些位模式条件，但数学意义上不是正整数幂。

位运算判断幂次时，先处理定义域，再处理位模式。

## 3 的幂和 4 的幂

`3` 的幂不能靠“只有一个二进制 1”判断，常见写法是最大 `3` 的幂对 `n` 取模：

```go
func isPowerOfThree(n int) bool {
	return n > 0 && 1162261467%n == 0
}
```

`4` 的幂既是 `2` 的幂，又要求唯一的 `1` 出现在偶数位：

```go
func isPowerOfFour(n int) bool {
	return n > 0 && n&(n-1) == 0 && n&0x55555555 != 0
}
```

## 易错点

- 运算符优先级容易误读，复杂表达式建议加括号。
- `n&(n-1)` 删除最低位的 `1`；如果结果为 `0`，说明原来只有一个 `1`。
- 幂次判断要先确认题目要求的数值范围。

## 复杂度

时间 `O(1)`，空间 `O(1)`。
