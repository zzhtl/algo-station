# 最少箭射气球：贪心训练题解

一支箭可以射爆所有包含该坐标的气球区间。要用最少箭，等价于选最少的点覆盖所有区间。

一句话记法：**当前箭放在最早结束区间的右端，覆盖尽量多后续区间。**

## Go 参考实现

```go
func findMinArrowShots(points [][]int) int {
	sort.Slice(points, func(i, j int) bool {
		return points[i][1] < points[j][1]
	})
	arrows := 0
	end := 0
	for i, p := range points {
		if i == 0 || p[0] > end {
			arrows++
			end = p[1]
		}
	}
	return arrows
}
```

## 为什么这样写

按右端点排序后，第一个还没被覆盖的区间右端最小。把箭射在它的右端，一定能覆盖它，并且不会比射在更左的位置差；更靠右还可能覆盖更多后续区间。

如果后续区间起点 `<= end`，它被当前箭覆盖；如果 `> end`，必须新增一支箭。

## 复杂度

- 时间复杂度：$O(n \log n)$。
- 空间复杂度：取决于排序实现。

## 易错点

- 重叠条件写成 `p[0] >= end`，但气球端点相等时一支箭可以同时射中。
- 按左端排序后难以保证最少箭。
- 空数组没处理；LeetCode 本题通常非空，工程代码要补。
- 用减法排序可能溢出。
