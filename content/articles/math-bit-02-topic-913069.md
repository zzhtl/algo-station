# 异或消去成对元素：数学与位运算训练题解

异或题最重要的性质只有三个：

```text
a ^ a = 0
a ^ 0 = a
a ^ b ^ a = b
```

只要题目保证“其他元素都成对出现，只有一个元素落单”，把所有数异或起来，成对元素会全部抵消，剩下的就是答案。

## Go 参考实现

```go
package main

func singleNumber(nums []int) int {
	ans := 0
	for _, x := range nums {
		ans ^= x
	}
	return ans
}
```

## 不变量

遍历到任意位置时，`ans` 等于“已经遍历过的所有数字的异或和”。由于异或满足交换律，数字出现的顺序不重要。出现两次的数字最终贡献为 `0`，只出现一次的数字最终保留下来。

这也是为什么这题不需要排序或哈希计数：题目给出的次数结构本身就是可抵消的。

## 缺失数字的同类写法

数组里包含 `[0,n]` 中缺失的一个数，也可以把索引和数值一起异或：

```go
func missingNumber(nums []int) int {
	ans := len(nums)
	for i, x := range nums {
		ans ^= i
		ans ^= x
	}
	return ans
}
```

完整集合里的数字和数组里实际出现的数字相互抵消，剩下的就是缺失值。

## 易错点

- 异或消去要求重复次数结构明确；如果元素出现三次，不能直接用这一招。
- 异或不是加法，不会产生进位，适合位级抵消。
- 初始化为 `0` 是因为 `0` 是异或的单位元。

## 复杂度

时间 `O(n)`，空间 `O(1)`。
