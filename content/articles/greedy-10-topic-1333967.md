# 买卖股票累计正收益：贪心训练题解

股票 II 允许多次交易，只要不同时持有多股。任何一段上涨区间的收益，都等于相邻正差值之和。

一句话记法：**涨了就吃这段差价，跌了就不参与。**

## Go 参考实现

```go
func maxProfit(prices []int) int {
	ans := 0
	for i := 1; i < len(prices); i++ {
		if prices[i] > prices[i-1] {
			ans += prices[i] - prices[i-1]
		}
	}
	return ans
}
```

## 为什么这样写

比如价格从 `1 -> 3 -> 6`，整体收益 `6-1` 等于 `(3-1) + (6-3)`。允许多次交易时，可以把一段连续上涨拆成每天买卖，收益不变。

下跌段不参与，因为它只会减少收益。

## 复杂度

- 时间复杂度：$O(n)$。
- 空间复杂度：$O(1)$。

## 易错点

- 把股票 I 的一次交易最低价逻辑套到股票 II。
- 以为必须真的记录每次买卖点。
- 含手续费或冷冻期仍然累加正收益，这时要用 DP。
- 空数组边界没处理；LeetCode 通常给非空。
