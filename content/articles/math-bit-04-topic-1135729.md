# 按位计数还原唯一数：数学与位运算训练题解

如果其他数字都出现两次，可以用异或抵消；如果其他数字都出现三次，异或就不够了。此时要把每一位单独看：出现三次的数字在每一位上的贡献都是 `3` 的倍数，模 3 后只剩唯一数字的那一位。

## 思路

对 32 个二进制位分别统计：

- 第 `bit` 位上有多少个数字是 `1`。
- `count % 3` 如果为 `1`，说明答案这一位是 `1`。
- 把这一位写回答案。

## Go 参考实现

```go
package main

func singleNumber(nums []int) int {
	ans := int32(0)

	for bit := 0; bit < 32; bit++ {
		count := 0
		for _, x := range nums {
			if (int32(x)>>bit)&1 == 1 {
				count++
			}
		}
		if count%3 != 0 {
			ans |= 1 << bit
		}
	}

	return int(ans)
}
```

## 为什么能处理负数

负数在补码表示下也只是 32 个二进制位。只要按固定 32 位统计，并把结果保存在 `int32` 中，最高位如果为 `1` 会自然表示负数。最后转回 `int` 即可。

如果直接用无界数学意义上的正负号去想，负数会很绕；按补码位模式处理更稳定。

## 易错点

- 不能只统计 31 位，否则负数最高位丢失。
- Go 中移位和类型有关，代码里统一转成 `int32` 可以避免符号位理解混乱。
- 如果其他数字出现 `k` 次，就把每位计数对 `k` 取模。

## 复杂度

固定 32 位，每位扫描一次数组，时间 `O(32n)`，空间 `O(1)`。
