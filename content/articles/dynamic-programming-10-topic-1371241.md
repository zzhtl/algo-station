# 编辑距离三来源转移：动态规划训练题解

编辑距离是二维字符串 DP 的代表。状态必须定义在前缀上：`dp[i][j]` 表示 `word1` 前 `i` 个字符变成 `word2` 前 `j` 个字符的最少操作数。

一句话记法：**看两个前缀末尾，相同看左上，不同看三种操作。**

## 状态定义

```text
dp[i][j] = word1[0:i] 变成 word2[0:j] 的最少操作数
```

边界：

- `dp[i][0] = i`：删掉 i 个字符。
- `dp[0][j] = j`：插入 j 个字符。

## Go 参考实现

```go
func minDistance(a string, b string) int {
	m, n := len(a), len(b)
	dp := make([][]int, m+1)
	for i := range dp {
		dp[i] = make([]int, n+1)
	}
	for i := 0; i <= m; i++ {
		dp[i][0] = i
	}
	for j := 0; j <= n; j++ {
		dp[0][j] = j
	}
	for i := 1; i <= m; i++ {
		for j := 1; j <= n; j++ {
			if a[i-1] == b[j-1] {
				dp[i][j] = dp[i-1][j-1]
			} else {
				dp[i][j] = min(dp[i-1][j], min(dp[i][j-1], dp[i-1][j-1])) + 1
			}
		}
	}
	return dp[m][n]
}
```

## 三种操作对应来源

- 删除 `word1[i-1]`：从 `dp[i-1][j] + 1` 来。
- 插入 `word2[j-1]`：从 `dp[i][j-1] + 1` 来。
- 替换末尾字符：从 `dp[i-1][j-1] + 1` 来。

末尾相同时，不需要操作，直接 `dp[i-1][j-1]`。

## 复杂度

- 时间复杂度：$O(mn)$。
- 空间复杂度：$O(mn)$，可压缩到 $O(n)$。

## 易错点

- `dp[i][j]` 的 i/j 和字符串下标 `i-1/j-1` 混淆。
- 边界没有初始化。
- 插入和删除来源写反虽然值可能一样，但理解会乱。
- Go 示例中的 `min` 需要自己实现或用辅助函数。
