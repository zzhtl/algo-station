# 低位技巧 x & -x：数学与位运算训练题解

`x & -x` 可以取出 `x` 的最低位 `1` 所代表的值，这个值也叫 `lowbit(x)`。

例如：

```text
x      = 10110000
-x     = 01010000  (补码意义下)
x & -x = 00010000
```

它的本质是补码取负会把最低位的 `1` 保留下来，并把它左边取反、右边全为 `0`。

## 统计 1 的个数

每次用 `x &= x-1` 删除最低位的 `1`，循环次数就是 `1` 的个数。

```go
package main

func hammingWeight(n uint32) int {
	count := 0
	for n != 0 {
		n &= n - 1
		count++
	}
	return count
}
```

`x & -x` 是取出最低位的 `1`，`x & (x-1)` 是删除最低位的 `1`。这两个技巧经常配套出现。

## lowbit 写法

```go
func lowbit(x int) int {
	return x & -x
}
```

树状数组里，`i += lowbit(i)` 表示跳到下一个覆盖区间，`i -= lowbit(i)` 表示向父区间回退。

## 易错点

- `lowbit(0) == 0`，循环里要避免 `i == 0` 时继续加减。
- 在 Go 中如果使用无符号类型，不能直接写 `-x`；可写 `x & (^x + 1)`。
- 判断奇偶只需 `x&1`，不需要 lowbit。

## 复杂度

按最低位 `1` 删除时，循环次数等于二进制中 `1` 的个数，最多是位宽；空间 `O(1)`。
