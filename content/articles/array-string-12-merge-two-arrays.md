# 双数组归并扫描：数组与字符串训练题解

双数组归并扫描来自归并排序：两个序列都已经有序时，当前最小候选一定在两个指针指向的位置之一。每轮比较这两个位置，然后移动贡献了答案的一侧。

这类题训练的是“同步推进两个序列”，不要退回去重扫。

## 适用场景

前提通常是两个数组、链表或区间列表已经有序，或者你可以先排序。

- 合并两个有序数组或链表。
- 求两个有序数组的交集。
- 求两个区间列表的交集。
- 归并过程中统计逆序对或中位数。

如果两个输入没有顺序，直接双指针没有意义；要么先排序，要么用哈希表。

## 图解思路

```mermaid
flowchart LR
  A[i 指向数组 A 当前元素] --> C{A[i] <= B[j]?}
  B[j 指向数组 B 当前元素] --> C
  C -->|是| D[取 A[i], i++]
  C -->|否| E[取 B[j], j++]
  D --> F[写入答案或更新状态]
  E --> F
  F --> C
```

区间交集也是同一个模型，只是比较的不再是单个值，而是两个区间：

- 交集左端是 `max(a.start, b.start)`。
- 交集右端是 `min(a.end, b.end)`。
- 谁的右端更小，谁就不可能再和后面的区间相交，移动谁。

## 手写步骤

1. 初始化两个指针 `i = 0, j = 0`。
2. 比较两个当前位置，决定谁贡献答案或谁可以丢弃。
3. 移动对应指针。
4. 如果需要保留重复值，逐个写入；如果需要去重，写入前检查上一个答案。
5. 扫描结束后，按题意处理剩余元素。

## Go 参考骨架

```go
func intersect(nums1 []int, nums2 []int) []int {
	sort.Ints(nums1)
	sort.Ints(nums2)
	i, j := 0, 0
	ans := []int{}
	for i < len(nums1) && j < len(nums2) {
		if nums1[i] == nums2[j] {
			ans = append(ans, nums1[i])
			i++
			j++
		} else if nums1[i] < nums2[j] {
			i++
		} else {
			j++
		}
	}
	return ans
}
```

## Rust 参考骨架

```rust
pub fn interval_intersection(first: Vec<Vec<i32>>, second: Vec<Vec<i32>>) -> Vec<Vec<i32>> {
    let (mut i, mut j) = (0usize, 0usize);
    let mut ans = Vec::new();
    while i < first.len() && j < second.len() {
        let lo = first[i][0].max(second[j][0]);
        let hi = first[i][1].min(second[j][1]);
        if lo <= hi {
            ans.push(vec![lo, hi]);
        }
        if first[i][1] < second[j][1] {
            i += 1;
        } else {
            j += 1;
        }
    }
    ans
}
```

## 为什么这样写

以 #350 两个数组的交集 II 为例，排序后较小的数不可能再和对方后面的更大数匹配，所以可以移动较小的一侧；相等时两边都移动，表示这两个实例已经配对。

以 #986 区间列表交集为例，右端更小的区间已经结束，它不可能再和后续区间产生交集，因此移动右端更小的一侧。

## 复杂度

- 两个输入已排序时，时间复杂度 $O(n + m)$。
- 如果需要先排序，整体是 $O(n \log n + m \log m)$。
- 空间复杂度取决于输出；原地合并可以做到 $O(1)$ 额外空间。

## 易错点

- 求交集 I 和交集 II 要区分：一个去重，一个保留重复次数。
- 合并到原数组时，如果从前往后写，会覆盖未读元素，应该从后往前。
- 区间交集中移动了左端更大的区间，而不是右端更小的区间，会漏掉后续交集。
- 两个数组都耗尽前的循环条件应该是 `i < n && j < m`，剩余元素要单独处理。

## 练习顺序

建议按这个顺序刷：#88, #350, #349, #986, #21, #4。

先练数组归并和交集，再做区间交集、链表归并，最后挑战两个有序数组中位数这种需要跳过一批候选的高级变体。
