# 快速幂二进制拆指数：数学与位运算训练题解

快速幂解决的是重复乘法太慢的问题。指数 `n` 可以拆成二进制，比如 `13 = 8 + 4 + 1`，所以：

```text
x^13 = x^8 * x^4 * x^1
```

每一轮只看指数最低位：如果这一位是 `1`，当前底数就要乘进答案；然后底数平方，指数右移。

## 不变量

循环中维护三个量：

- `ans`：已经确认需要乘入的幂次贡献。
- `base`：当前二进制位对应的底数，依次是 `x, x^2, x^4, x^8...`。
- `exp`：还没处理的指数位。

每轮处理完最低位后，`base` 平方，`exp` 右移一位，问题规模减半。

## Go 参考实现

```go
package main

func myPow(x float64, n int) float64 {
	exp := int64(n)
	if exp < 0 {
		x = 1 / x
		exp = -exp
	}

	ans := 1.0
	base := x
	for exp > 0 {
		if exp&1 == 1 {
			ans *= base
		}
		base *= base
		exp >>= 1
	}
	return ans
}
```

## 为什么先转成 int64

`n` 可能是最小整数，直接取负会溢出。把 `n` 转成 `int64` 后再处理负指数，可以避开这个边界。

负指数不需要单独设计循环，只要利用 `x^-n = (1/x)^n`，把底数反转、指数变正即可。

## 易错点

- `base *= base` 要每轮都做，不管当前位是不是 `1`。
- 判断最低位用 `exp&1 == 1`，然后右移 `exp >>= 1`。
- 浮点结果允许误差，重点是指数边界和时间复杂度。

## 复杂度

指数每轮减半，时间 `O(log |n|)`，空间 `O(1)`。
